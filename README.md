# Medical-Store-Management-System-Cpp
A professional Medical Store Management System built with C++ and Qt 6. Features include real-time inventory tracking, secure login, automated backups, and a POS billing system with printable receipts. Uses binary file handling for data persistence. Fully deployed for Windows with a modern, responsive UI.
